CLI Send serial commands directly

Use the same commands as the serial control for the cube. Please refer to the CUBE4 RGB Cube Command List.

  • Quick Reference
  • all [colour];
  • shift [axis] [direction];
  • set [location] [colour];
  • next [colour];
  • setplane [axis] [offset] [colour];
  • copyplane [axis] [source offset] [destination offset];
  • moveplane [axis] [source offset] [destination offset] [colour];
Don't forget to finish the command with a semi-colon (;)
